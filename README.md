# AI Resume Builder

## About the Project

This is a full-stack AI Resume Builder that I am building as a personal project to improve my software development skills.

The goal of this project is to create a web application where users can create professional resumes, improve their resume content using AI, and download ATS-friendly PDF resumes.

This project is also helping me learn full-stack development using React, Flask, PostgreSQL, and Generative AI.

---

## Features (Planned)

- User Authentication
- Create and manage multiple resumes
- Resume Editor
- AI-generated Professional Summary
- AI Bullet Point Improvement
- ATS Resume Checker
- PDF Export
- Responsive UI
